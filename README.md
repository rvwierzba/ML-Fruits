# Aprendendo - WebAPI & WebApp
## WebApp Blazor simples feito em C# .NET 7 que consome WebAPI que realiza o CRUD de frutas (CREATE, UPDATE and DELETE);
### Pode ser entendida como uma estrutura de Microsserviços, sendo que a Aplicação como um serviço (front-end) consome uma api como outro serviço (back-end);
#### Baseado no Lab do Microsoft Learn em: https://microsoftlearning.github.io/APL-2002-develop-aspnet-core-consumes-api/
